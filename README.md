# GameRecord
This project is a video game library web application that allows users to discover, organize, and track their video games in a complete and centralized way.

Games will be fetched from an external API to provide a rich and continuously updated catalog. Users will be able to search for games and add them to different custom collections such as wishlist, backlog, currently playing, or completed games.

In addition to organizing their library, users will be able to leave reviews and ratings for the games they have played, helping them keep track of their experiences and share their feedback.

The backend will be built with Spring Boot as a way to further improve my backend development skills and deepen my experience within the Spring ecosystem. The frontend will be developed with React to gain more practice with modern frontend development.

The overall goal of this project is to strengthen my full-stack development skills by combining a Java/Spring backend with a React frontend in a realistic, scalable, and practical application.
